# Skill Exchange Platform

A Java-based web application that enables users to share their skills, browse skills offered by others, and send or manage skill exchange requests.

## Features

- User Registration & Login
- Profile Management
- Add, Edit, View & Delete Skills
- Browse Skills
- Send Skill Exchange Requests
- View Sent Requests
- Accept/Reject Received Requests

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- Apache Tomcat 10
- Eclipse IDE
